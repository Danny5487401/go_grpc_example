# AMQP（Advanced Message Queuing Protocol)
就像java的Java消息服务—— JMS，负责定义标准，遵守JMS标准的消息队列就可以再java平台使用。


不同于JMS将平台限制为java， AMQP 没有限制平台，任何遵守 AMQP 协议的消息队列，可以再任由语言平台上使用，甚至是发送消息和消费消息的是两个语言平台
